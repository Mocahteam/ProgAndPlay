# Prog&amp;Play for Scratch 2.0
* Open ProgAndPlayANE project with FlashDevelop (tested with version 5)
* Build the project => It produces ProgAndPlayANE.swc file
* Switch to lib directory and use make to build ane file
* Use make install to install ane on Scratch project
* Copy/paste pp-client.dll to the in the same folder as the adl.exe file (something like C:\Users\<UserName>\AppData\Local\FlashDevelop\Apps\flexairsdk\4.6.0+19.0.0\bin)
* Open Scratch project with FlashDevelop
* Build the project and enjoy
