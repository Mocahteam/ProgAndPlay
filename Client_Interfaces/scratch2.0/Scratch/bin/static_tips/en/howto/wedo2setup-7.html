Troubleshooting

If it's not working:

  • Make sure you have installed the Device Manager.
  • Make sure your WeDo 2.0 is turned on and ready to connect.
  • Turn on Bluetooth on your computer.
  • Make sure you have a Mac with Bluetooth Low-Energy (see below).
  • Close any other tabs or windows running with Scratch.
  • Restart your browser.
  • Restart the Device Manager (go to your Applications folder and open the Scratch Device Manager).
  • If you have multiple devices, make sure you are connected to the right WeDo 2.0. You can connect and disconnect using the toggle buttons in the Device Manager interface.
  • Remove any previous versions of the Device Manager.

Does your Mac have Bluetooth Low Energy?

Here's how to check if your Mac can use the LEGO WeDo 2.0 extension.

1. Go to the Apple menu and select 'About This Mac'

2. Click 'System Report', then 'Bluetooth.'

3. Look to see if it says: 'Bluetooth Low Energy Supported: Yes'

If it says 'No' then you can use a USB Bluetooth adapter.
