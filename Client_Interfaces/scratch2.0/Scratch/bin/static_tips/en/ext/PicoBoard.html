PicoBoard Extension

This extension works with the PicoBoard (previously known as Scratch Sensor Board).

IMPORTANT: Install the Plug-In

All hardware extensions need a plug-in to work. If the plug-in is not already isntalled, check the Extensions page for instructions.

Blocks:
Components:

The PicoBoard incorporates a light sensor, sound sensor, a button and a slider, as well as 4 additional inputs that can sense electrical resistance via included cables.

To learn more, see the PicoBoard page on the Scratch wiki.
